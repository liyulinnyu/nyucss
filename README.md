# nyucss
NYU Cascading Style Sheets

## nyu-carousel

更加方便美观地做出适配所有屏幕的轮播图效果，非等比例式适配浏览器伸缩。您只要设置三个值即可：
1. 整体轮播屏的高度（如果需要的话，默认为最小高度）
2. 要轮播的图片
3. 左右控制按钮填充样式
